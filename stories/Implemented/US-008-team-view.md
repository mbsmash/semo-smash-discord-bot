# US-008: View Teams

## Summary

As a server member, I want to view the list of teams so I can see who's on them.

## Acceptance Criteria

**Given** teams exist with players on them

**When** a server member uses the command `/teams` with no additional arguments

**Then** the bot replies with a list of all of the teams currently saved

**And** the list contains the following information about each team:

    - Team Name
    - Team Score
    - List of team members
